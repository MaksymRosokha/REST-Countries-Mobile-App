# REST Countries App

A mobile application that uses the REST Countries API to display information about countries around the world. Users can browse a list of countries and view detailed information for each selected country.

## Features

* Browse a list of countries
* View detailed country information
* Fetch data using multiple REST API endpoints
* Offline access through local database caching
* Loading state handling with user feedback
* Error handling with clear and user-friendly messages

## Architecture

The application follows an offline-first approach by storing API responses in a local database. Cached data remains available even when the device has no internet connection.

## API

* REST Countries API: https://restcountries.com/

## Requirements Covered

* At least two screens (Countries List and Country Details)
* At least two REST requests (list and details)
* Offline functionality using a local database
* Loading state management
* Error handling and user notifications
